# Princess_Bari_K-Culture_Metaverse_Game
본 프로젝트는 한국 전래동화 바리공주 설화를 바탕으로 제작한 체험형 메타버스 VR 퀘스트 게임입니다.  Unity와 Viven SDK를 활용하여 실제 플레이 가능한 맵을 구현하였으며, 마고 할미의 시험, 지옥의 문, 서천 꽃밭이라는 3단계 퀘스트 구조를 통해 사용자가 스토리를 직접 체험하도록 설계했습니다.  게임 전반에 한국 민화풍 비주얼, 전통적 공간 구성, 신나면서도 한국적인 배경 음악을 적용하여 K-컬처와 전래동화를 자연스럽게 즐길 수 있는 메타버스 콘텐츠를 목표로 합니다.  본 프로젝트는 K-컬처 확산을 주제로 한 해커톤 출품작이자, 포트폴리오용 결과물입니다.
